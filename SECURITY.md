## Security

Microsoft takes the security of our software products and services seriously, which includes all source code repositories managed through our GitHub organizations.

If you believe you have found a security vulnerability in this project, do not open a public GitHub issue.

Please report it through the Microsoft Security Response Center (MSRC):
- MSRC portal: https://msrc.microsoft.com/report/vulnerability
- MSRC information: https://www.microsoft.com/msrc

Please include as much of the following information as possible:
- Type of issue
- Full paths of affected files or components
- Steps to reproduce the issue
- Proof-of-concept or sample input, if available
- Potential impact assessment

Microsoft follows coordinated vulnerability disclosure practices and will investigate reports submitted through MSRC.
